## 02 — MANUAL

$ man nexus

NAME — nexus — assemble, nest and broadcast knowledge. The flags you'll actually use:

--blocks
Documents are ordered blocks: heading, paragraph, list, image, video, file, embed, divider. Reorder by dragging; insert with `/`.
--nest=∞
Infinite hierarchy. No max depth, no "you can't put a page there." The tree is the only data model.
--realtime
WebSocket subscriptions broadcast block_added, block_updated, cursor_position. Edit together, conflict-free.
--import=notion
Pipe a Notion URL → fetch HTML → parse DOM → emit clean Nexus blocks. h1→heading, p→paragraph, img→image.
--events
Every mutation appended to an event log: DOCUMENT_CREATED, FOLDER_MOVED… audit, replay, automate.
--role=admin|editor|viewer
Permissions scoped per business. RLS at the source, not bolted on the client.
