▮ The Rack

Six modules in the rack.

Each capability is a module you can patch into any other. Build the system that sounds like your mind.

VCO · BLOCKS

Block Editor

Oscillate between text, image, video and embed. Reorder the waveform by dragging.

SEQ · NESTING

Infinite Nesting

Sub-patch within sub-patch. The rack has no fixed number of rows.

CLK · SYNC

Real-Time

One master clock. Every collaborator's cursor stays in sync, sample-accurate.

IN · IMPORT

Notion Import

External input: a URL is sampled and re-voiced as clean Nexus blocks.

MEM · ASSETS

Asset CDN

Sample memory for images and video, streamed from the nearest edge.

GATE · ROLES

Access Roles

Admin, editor, viewer — gates that decide which signals each patch can pass.

▮ Step Sequencer

Programme what fires when.

A content calendar laid out as a step sequencer — each lit step a node, triggered on the day it ships.
