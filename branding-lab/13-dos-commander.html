INSTALLED MODULES
Six TSR programs loaded into memory, ready on command.
[BLOCKS.SYS]

Documents built from ordered blocks — text, image, video, embed. Reorder with +/- keys.

[NEST.SYS]

Subdirectories without a depth limit. The tree fills the whole volume.

[SYNC.COM]

Real-time multi-user editing. Live cursors broadcast over the channel.

[IMPORT.BAT]

Pipe a Notion URL → parse → write clean Nexus blocks to disk.

[ASSETS.DRV]

Image and video device, served fast from the CDN cache.

[ROLES.CFG]

ADMIN / EDITOR / VIEWER permissions, set per business volume.
