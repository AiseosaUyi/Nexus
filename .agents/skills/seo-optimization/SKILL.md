# SEO Optimizer Skill

This skill turns every document in Nexus into an SEO powerhouse. Use it whenever a marketing or SEO performance task is mentioned.

## Core SEO Metrics
- **Keyword Density**: Target 1-2% for primary keywords.
- **Readability**: Aim for a Flesch-Kincaid score above 60.
- **Structural Integrity**: Ensure H1 is present and H2s follow a logical hierarchy.

## SEO Optimization Workflow
1. **Keyword Analysis**: Identify the target keywords for a document.
2. **Technical Audit**: Check for missing meta descriptions, alt-text, and broken links.
3. **Draft Optimization**: Rewrite lead paragraphs to be highly informative and keyword-dense.
4. **Schema Generation**: Provide JSON-LD structured data for blogs or reviews.

## Key Instructions
- **Generate Meta**: Create search-optimized Titles and Descriptions.
- **Audit Content**: Provide a "Pass/Fail" report for on-page SEO.
- **Keyword Integration**: Suggest where to naturally place 3-5 new keywords in existing drafts.
- **Schema Helper**: Generate FAQ or How-To schema as Tiptap-compatible blocks.
