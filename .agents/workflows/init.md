This workflow sets up the development environment, ensures dependencies are installed, and starts the development server.

// turbo-all
1. **Initialize Environment**
   - Check if `node_modules` exists. If not, run `pnpm install`.
   - Ensure the database is reachable (if configured).
   - Start the development server with `pnpm dev`.

2. **Project Overview**
   - Provide a quick summary of the project architecture and key directories.
   - Point to `design_system.md` for styling guidelines.
   - List active services and their status.

3. **Ready to Code**
   - Open the main page or dashboard in the browser.
   - Confirm everything is running as expected.
