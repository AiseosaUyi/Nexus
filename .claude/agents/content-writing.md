# Content Writing Skill

This skill transforms Nexus into a professional-grade content engine. Use it whenever the user is building documents that require narrative flow or a specific brand voice.

## Brand Voice Principles
- **Clarity First**: Avoid fluff. Use active voice and strong verbs.
- **Structural Integrity**: Every document must have a clear H1 title, a concise executive summary, and well-organized sub-headings (H2/H3).
- **User-Centric**: Write for the reader. What problem are we solving?

## Writing Workflow
1. **Outline**: Before drafting, provide a block-by-block outline.
2. **Drafting**: Use the Nexus block system (H1, H2, Paragraph, Lists).
3. **Refinement**: Audit the document for "readability" and "flow".

## Key Commands
- **Draft Blog**: Generate a 800-1200 word post with optimized h-tags.
- **Document Summary**: Distill complex pages into 3 high-impact bullet points.
- **Tone Shift**: Rewrite content to be more "Professional", "Casual", or "Technically Dense".

## Content SEO Baseline
Ensure every content piece includes:
- A clear Meta Description suggestion (max 160 chars).
- At least 3 primary keyword integrations within the first 200 words.
- Descriptive alt-text placeholders for images.
