# LinkedIn — playbook (kind: content)

LinkedIn is where higher-budget clients and companies find designers. Content builds authority;
DMs and comments turn into real gigs.

## Read (each run)
- Notifications → comments on your posts, meaningful reactions, profile views worth a follow-up.
- Messaging → inbound from recruiters/founders. Scam-filter ("recruiter" + shortened link = flag).

## Draft (content engine, shared idea bank with Twitter)
- Post ideas: case study of a project (problem → what you did → result), a design lesson, a
  before/after, an "open to select projects" post. Same idea bank as Twitter, reformatted.
- LinkedIn format: strong first line (it's the hook before "…more"), short paragraphs, 1 image or
  a carousel/video, 3-5 relevant hashtags, a soft CTA.
- Comment/reply drafts to legit inbound.

## What good looks like on LinkedIn
- Story + outcome beats pure aesthetics. Numbers/results land ("cut signup drop-off," "shipped in 2 weeks").
- Carousels and native video outperform bare links.
- Professional but human; first line does the work.

## Health inputs
- Posting cadence, engagement rate, headline + "open to work" set, profile completeness, connection
  growth. Top fix is usually: post the case study in the calendar / sharpen the headline.

## Execute (after approval)
- Post: Start a post → paste approved copy → attach approved media (image/carousel/video) →
  post → `cc_mark_posted` + URL.
- Reply/DM: open thread → paste approved text → send → `cc_mark_sent`.
