# Contra — playbook (kind: both)

Contra is commission-free freelance: portfolio + projects + messages + a discovery feed/community.
Inbound comes via **messages/inquiries** and **project opportunities**; visibility comes from a
strong portfolio and being active in the feed.

## Read (each run)
- Inbox → client messages, inquiries. Scam-filter each.
- Opportunities / projects feed → briefs matching Aise's services.
- Community/feed posts worth engaging (Contra rewards active profiles).

## Draft
- Inquiries: proposal, confirm scope + budget, keep on-platform.
- Opportunities: tailored pitch referencing the brief + a relevant portfolio piece.
- Feed engagement: short, genuine comments on relevant posts (capture as type=comment).

## What good looks like on Contra
- Portfolio filled with project cards (cover + outcome), services + rates listed, "available" on.
- Regular feed activity (a post or thoughtful comments) keeps the profile surfaced.
- This is the platform where Aise often triggers you manually: "go to this Figma, export it,
  upload it to Contra with this copy." Follow the exported-frames → upload flow.

## Health inputs
- Portfolio depth, services/rates set, availability on, feed activity, response speed.
  Top fix is usually: add a project card or post to the feed this week.

## Execute (after approval)
- Message/pitch: open thread → paste approved text → send → `cc_mark_sent`.
- Portfolio/project upload: Aise points to the Figma/exported files; you create the project card
  with approved cover + copy → publish → `cc_mark_posted` + URL.
- Feed post: paste approved copy + media → post → `cc_mark_posted`.
